# sorting-visualization-js

Simple array sorting visualization.<br>
GitHub Pages: https://andrground-gg.github.io/sorting-visualization-js/<br><br>
Algorithms available: 
<ul>
  <li>Bubble Sort</li>
  <li>Selection Sort</li>
  <li>Quick Sort</li>
  <li>Merge Sort</li>
  <li>Radix Sort</li>
</ul>
